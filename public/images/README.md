# Imágenes de Perfil

## 📸 Cómo agregar tu foto de perfil

1. **Agrega tu foto** en esta carpeta con el nombre:
   ```
   francisco-higuera-photo.jpg
   ```

2. **Formatos recomendados:**
   - **Dimensiones**: 400x400px (cuadrada)
   - **Formato**: JPG o PNG
   - **Tamaño**: Máximo 200KB
   - **Estilo**: Foto profesional, fondo neutro, rostro visible

3. **Ubicación del archivo:**
   ```
   /public/images/francisco-higuera-photo.jpg
   ```

## 🎨 Características del Avatar

- **Circular**: Se muestra automáticamente en un círculo
- **Fallback**: Si la foto no carga, muestra tus iniciales (FH)
- **Responsive**: Se adapta a diferentes tamaños
- **Efectos**: Hover suaves y transiciones elegantes

## 🔄 Actualización

Para cambiar tu foto:
1. Reemplaza el archivo `francisco-higuera-photo.jpg`
2. Refresca la página web (Ctrl+F5 o Cmd+R)
3. La nueva foto aparecerá automáticamente

## 📱 Nota

El avatar se muestra en:
- **Header principal**: Versión pequeña (36px)
- **Menú de perfil**: Versión mediana (48px)
- **Dropdown**: Versión grande con información completa
